¿Qué puede realizar aquí?



Geo-EC-Maps, nace de la necesidad de brindarle a la comunidad ecuatoriana una alternativa de software libre que les permita realizar la configuración de mapas que pueden ser implementados dentro de diferentes e-commerce de forma gratuita. Una de las principales características de este proyecto es que al ser de código abierto el usuario tiene la oportunidad de descargar el mapa en el cual ha trabajado y colocarlo de manera libre en su negocio digital.



Tecnologías Utilizadas

Geo-EC-Maps hace uso de las siguientes tecnologías:

JavaScript

HTML 5

CSS 3

Leaflet

OpenStreetMap

GeoJSON
